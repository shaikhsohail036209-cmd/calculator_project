# Modern GUI Calculator

## Overview

This is a responsive calculator web application built using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations and provides a clean, user-friendly interface.

## Features

* Addition, Subtraction, Multiplication, and Division
* Parentheses Support
* Decimal Calculations
* Clear Button (C)
* Backspace Button (⌫)
* Keyboard Support
* Responsive Design
* Error Handling for Invalid Expressions

## Technologies Used

* HTML5
* CSS3
* JavaScript
